Pizza Luiza - проект академии верстки GLO. На основе, ими предоставленного дизайна и базы данных JSON,
На языке Java Script я реализовал:


- Поиск товаров на сайте

- Cортировку товаров по категориям 

- Работу корзины

* Добавление товаров в корзину

* Удаление товаров из каризны

* Расчет стоимости добавленных товаров

- Возможность выбирать понравившиеся товары и доьавлять их в отдельный "WishList"
